Publié le Min 29 Januari 2017 par Andy Huynh dans «Online Learning». Mots-clés: Analysis

Genereal Framework

  • predict a vector $w_t \in S$

  • receive a convex loss function $f_t: S \rightarrow R$

  • suffer loss $f_t(\mathbf{w}_t)$

Follow the Regularized Leader Mantra

$$\mathbf{w}_{t+1} = arg \min_w \bigg\{g_{1:t} \cdot w + \alpha_{1:t}\psi(w) + r_{0:t}(w)\bigg\}$$

FTRL-proximal¶

Formulation

$$w^{t+1} = \underset{w}{\mathrm{argmin}} \bigg\{g^{(1:t)} \cdot w + \lambda_1\|w\|_1 + \lambda_2\|w\|_2^2 + \frac{1}{2}\sum_{s=1}^t \sigma^{s}\|w-w^{(s)}\|_2^2\bigg\}$$$$\text{where } \sigma^{(s)}_i = \frac{1}{\alpha}\bigg(\sqrt{\sum_{j=1}^s (g_i^{(j)})^2} - \sqrt{\sum_{j=1}^{s-1} (g_i^{(j)})^2}\bigg)$$
